# 基于OpenCV全景图像拼接系统 python1038

本项目是一款基于Python和OpenCV技术的全景图像拼接系统，采用MySQL进行数据存储，配备简洁的前端设计，实现了用户友好的图像拼接与管理功能。

## 技术栈

- **后端语言**: Python 3.7
- **数据库**: MySQL
- **编译器**: PyCharm
- **前端**: 简洁的网页设计

## 功能模块

系统定义了两种用户角色：普通用户和管理员用户。

### 管理员用户

- 个人信息管理：维护个人密码。
- 图片列表管理：对已拼接全景图片的信息进行维护和管理。

### 普通用户

- 网站首页浏览：查看系统功能，享受简洁的界面设计。
- 图片拼接模块：注册并登录后，可使用在线图片拼接功能。

## 系统角色

- 普通用户：需注册后方可登录系统，进行图片拼接等操作。
- 管理员用户：预设定账户，拥有用户管理的权限。

## 运行环境

- Python 3.7 或以上版本
- MySQL 数据库
- 兼容主流的现代浏览器

## 部署步骤

1. 安装 Python 3.7 或更高版本。
2. 安装 MySQL 数据库。
3. 克隆或导入项目代码至本地开发环境。
4. 根据环境配置数据库连接。
5. 启动后端服务。
6. 部署前端代码至Web服务器。
